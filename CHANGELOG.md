## [v0.1.0] - 2026-01-12

### Added
- Initial release of Copy4LM plugin for IntelliJ IDEA
- Add support for copying file contents from open tabs or selected files
- Add configurable header and footer formatting for copied text
- Add configurable file size limit setting (default 500 KB)
- Add unit tests for core use cases and domain logic
- Add automated release workflow via GitHub Actions

### Changed
- Update default header and footer formats to use triple backticks for better code block rendering

### Internal
- Refactor codebase to hexagonal architecture (ports and adapters)
- Rename plugin package and ID from `copyfilecontent` to `copy4lm`
- Update test dependencies (JUnit Jupiter 5.14.1, MockK)
